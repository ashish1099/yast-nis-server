NIS server module implementation

Files

The main module file is nis_server.ycp. It simple includes appropriate include files and defines sequence for wizard sequencer. Include files (the routines description can be found at autodocs):

ui.ycp
is the main user interface file that defines some more dialogs (reading, saving, pop-ups) and includes all "wizard dialogs".
io.ycp
contains all read/write routines and also some default values for all user_settings values
routines.ycp
useful routines within the NIS server configuration
what.ycp
first wizard dialog (what to configure)
slave.ycp
slave configuration dialog
master.ycp
master configuration dialog
details.ycp
details for master configuration dialog
slaves.ycp
slaves for master configuration dialog (optional)
maps.ycp
maps wizard dialog
securenets.ycp
securenets dialog

There are currently used these agents, for accessing files:

  • rcnf for accesing /etc/rc.config and /etc/rc.config.d/ypserv.rc.config
  • ag_yp_makefile perl agent for accessing /var/yp/Makefile
  • var_yp_securenets.scr any agent description for /var/yp/securenets
  • var_yp_ypservers.scr any agent description for /var/yp/ypservers
